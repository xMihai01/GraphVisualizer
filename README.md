# GraphVisualizer

- Click to add new node
- Drag to add an Arc between 2 nodes
- Click "Edit" to move nodes
- Click "Graf orientat" to change Graph mode
- Click "ChangeDiameter" to change diameter
- Click "DrawNodes" to generate random nodes
